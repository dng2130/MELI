# Mercado Libre Seller Strategy API

This project implements an API that generates commercial strategies for Mercado Libre sellers using clustering analysis and an LLM.

## Project Overview

The solution identifies seller segments using clustering techniques and generates tailored commercial strategies for each seller using a language model.

The API receives a `seller_nickname`, retrieves the seller data, identifies its cluster, and generates a strategy recommendation.

## Tech Stack

- Python
- FastAPI
- Pandas
- OpenAI API
- Machine Learning (Clustering)
